# toolbox

VBA / PowerShell 単発マクロ・スクリプト集。

| ツール | 説明 |
|--------|------|
| [ExcelDataExtractor](ExcelDataExtractor/) | ADO 経由で複数 Excel ファイルからセルデータを一括抽出 |
| [OutlookDraftCreator](OutlookDraftCreator/) | Excel シートから一括で Outlook 下書きメールを作成 |
| [VBAPasswordRemover](VBAPasswordRemover/) | VBA プロジェクトのパスワード保護を解除 (VBA / PowerShell) |
| [VBACodeExtractor](VBACodeExtractor/) | Excel を開かずに VBA コードをテキスト抽出 (PowerShell) |

詳細は各フォルダの README を参照。
